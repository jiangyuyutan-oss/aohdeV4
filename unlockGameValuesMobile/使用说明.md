# 解锁 GameValues 移动端加载 (unlockGameValuesMobile)

移除原作者设置的移动端限制，让 Android 设备也能加载自定义 GameValues 配置文件。

## 问题

原作者在 `Menu_InitGame.smali` 中对所有自定义 GameValues `.txt` 文件添加了 `getIsDesktop()` 平台检查，导致在移动端（Android）这些配置文件永远不会被加载。

## 解锁的文件

| 配置文件 | 功能 |
|---------|------|
| `ENDA.txt` | END 相关配置 |
| `ORHS.txt` | ORHS 相关配置 |
| `EAPWS.txt` | EAPWS 相关配置 |
| `EACDG.txt` | EACDG 相关配置 |
| `DJE.txt` | DJE 相关配置 |
| `UCSH.txt` | UCSH 相关配置 |
| `SANDISINF.txt` | SANDISINF 相关配置 |
| `TDSX.txt` | TDSX 相关配置 |
| `WVFAL.txt` | WVFAL 相关配置 |
| `GLDRCA.txt` | GLDRCA 相关配置 |

## 修改内容

从 `Menu_InitGame.smali` 中移除了 10 处 `getIsDesktop()` 检查，共删除 66 行代码。

## 使用方式

### 方式一：懒人包 (lazy/)

直接替换 `age/of/civilizations2/jakowski/lukasz/Menus/Menu_InitGame.smali` 文件。

### 方式二：手动修改 (mod/)

参考 `Menu_InitGame.diff` 文件，手动修改原文件中的 10 处 `getIsDesktop()` 检查。

每处需要删除的代码模式：
```smali
invoke-static {}, Lage/of/civilizations2/jakowski/lukasz/CFG;->getIsDesktop()Z
move-result vX
if-eqz vX, :cond_XXX
```

## 注意事项

- 此修改仅移除平台限制，实际生效还需要在游戏的 `game/gameValues/` 目录下放置对应的 `.txt` 配置文件
- 配置文件格式为 JSON 风格
- 如果同时使用 `customImages` 插件，建议使用 `customImages` 中的合并版本，避免冲突

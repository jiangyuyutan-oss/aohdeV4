# CTAP Mod 插件

CTAP (Chinese Text Adaptation Plus) - 中文文本适配增强 Mod，为《文明时代 2》(Age of History II) 提供中文显示优化、数字简化、日期格式化及领导人肖像悬浮窗等功能。

---

## 功能总览

| 功能 | 说明 | 适用场景 |
|------|------|---------|
| 中文文本适配 | 解决中文在游戏 UI 中的显示问题，支持 CJK 字符分词 | 中文语言模式下 |
| 中文数字简化 | 将大数字显示为"万"、"亿"格式（替代 K/M/B） | 人口、金钱等数值显示 |
| 中文日期格式 | 游戏日期使用中文格式显示 | 日历/时间显示 |
| UI 缩放适配 | 自动缩放中文 UI 元素防止溢出 | 各种菜单界面 |
| 领导人肖像悬浮窗 | 点击领导人时在悬浮窗中显示肖像大图 | 领导人信息查看 |

---

## 技术原理

### Mixin 注入机制

CTAP 基于 `team.rainfall.finality.luminosity2` 注入框架，通过 **Mixin 注解**实现对游戏原有方法的覆盖。无需修改原版 smali 文件，通过运行时替换实现功能增强。

```
┌─────────────────────────────────────────────────────┐
│                    游戏原始代码                       │
│  age.of.civilizations2.jakowski.lukasz.*            │
│                                                     │
│  Core.getHover_LeaderOfCiv()  ← 原始领导人悬浮窗方法   │
│  CFG.getNumber_SHORT()        ← 原始数字简化方法       │
│  GameCalendar.getCurrDate()   ← 原始日期获取方法       │
│  TouchManager.updateEnableScaling() ← UI 缩放方法     │
└──────────────────────┬──────────────────────────────┘
                       │ 运行时注入（Mixin）
                       ▼
┌─────────────────────────────────────────────────────┐
│                  CTAP 注入层                          │
│  team.rainfall.ctap.*                               │
│                                                     │
│  MixinCore       → 覆盖 Core         (领导人悬浮窗增强) │
│  MixinCFG        → 覆盖 CFG          (中文数字简化)     │
│  MixinGameCalendar → 覆盖 GameCalendar (中文日期格式)  │
│  MixinTouchManager → 覆盖 TouchManager  (UI 缩放)    │
│  MixinMapBG      → 覆盖 MapBG         (背景处理)      │
│  MixinCFG        → 覆盖 CFG           (数字/分词)      │
└─────────────────────────────────────────────────────┘
```

### 各模块工作原理

#### 1. MixinCore - 领导人悬浮窗增强

覆盖 `Core.getHover_LeaderOfCiv(int civID)` 方法，重建悬浮窗内容列表：

```
原始流程:
  [领导人名称] → [旗帜图标] → [宗教图标] → [分割线] → [出生/死亡日期] → ...

注入后流程:
  [领导人名称]
  [领导人肖像]  ← 新增：读取 game/leadersIMG/{image}_SL.png
  [旗帜图标] + [宗教图标]
  [分割线]
  [出生日期] + [时代信息]
  [死亡日期] (如果适用)
  [详细信息行]
  ...
```

**肖像加载逻辑：**
```
1. 读取 leaderData.getImage() → 如 "trump.png"
2. 去除扩展名 → "trump"
3. 拼接 _SL 后缀 → "trump_SL.png"
4. 完整路径 → "game/leadersIMG/trump_SL.png"
5. 通过 IMGManager.addIMG() 加载为纹理
6. 创建 ME_Hover_2Type_Image_Big 显示在悬浮窗第二行
```

#### 2. MixinCFG - 中文数字简化

覆盖 `CFG.getNumber_SHORT(int nValue)` 方法，判断语言后切换简化规则：

```
英文模式 (原版):
  1000 → "1K"     1000000 → "1M"     1000000000 → "1B"

中文模式 (CTAP):
  10000 → "1万"    100000000 → "1亿"    1000000000000 → "1万亿"
```

判断条件：`CFG.settingsGD.LANG_TAG` 包含 `"cn"` 时启用中文模式。

#### 3. MixinGameCalendar - 中文日期格式

覆盖 `GameCalendar.getCurrDate()` 方法：

```
英文模式 (原版):
  "January 1, 1444"

中文模式 (CTAP):
  "1444年1月1日"
```

#### 4. MixinTouchManager - UI 缩放适配

覆盖 `TouchManager.updateEnableScaling()` 方法，在游戏运行时动态启用 UI 缩放，确保中文文本在各种分辨率下正常显示。

#### 5. CTAP - CJK 文本处理工具

提供中文/日文/韩文 (CJK) 字符识别和分词功能：

- `isCJK(char)` - 判断字符是否属于 CJK 字符集（汉字、平假名、片假名、韩文）
- `tokenize(String)` - 将混合文本按 CJK 字符分词（每个汉字独立处理，英文单词保持完整）
- `hasGlyph()` / `getGlyph()` - 检测字体是否包含特定字符的字形
- `getGlyphLayoutData()` - 线程安全地获取文字布局数据

#### 6. NumberShortenUtil - 中文数字转换

实现中文数字简化算法，支持：
- 万 (10^4)、亿 (10^8)、万亿 (10^12) 等单位
- 保留小数精度（如 "1.5亿"）

#### 7. DateUtil - 中文日期工具

提供中文日期格式化方法。

#### 8. MixinMapBG - 背景处理

处理地图背景相关的适配逻辑。

## 文件结构

```
CTAP/
├── lazy/                          # 懒人包（完整注入，一键覆盖）
│   └── team/rainfall/ctap/
│       ├── CTAP.smali             # CJK 文本处理工具
│       ├── CTAP$$ExternalSyntheticLambda0.smali  # Lambda 辅助类
│       ├── DateUtil.smali         # 中文日期工具
│       ├── MixinCFG.smali         # CFG 注入（数字简化）
│       ├── MixinCore.smali        # Core 注入（领导人肖像）
│       ├── MixinGameCalendar.smali # 日历注入（中文日期）
│       ├── MixinMapBG.smali       # 背景注入
│       ├── MixinTouchManager.smali # 触摸注入（UI 缩放）
│       └── NumberShortenUtil.smali # 中文数字转换
│
├── mod/                           # 模组包（仅修改文件）
│   └── team/rainfall/ctap/
│       └── MixinCore.smali        # 唯一新增功能文件（领导人肖像）
│
├── compile/                       # 编译产物（参考用）
│   └── out/smali/team/rainfall/ctap/
│       └── MixinCore.smali
│
└── 使用说明.md                    # 本文档
```

### lazy/ vs mod/ 区别

| | lazy/ (懒人包) | mod/ (模组包) |
|---|---|---|
| 包含文件 | CTAP 全部 9 个文件 | 仅 MixinCore.smali |
| 适用人群 | 新手用户 | 有经验的 Modder |
| 安装难度 | 低（覆盖整个目录） | 中（需要理解注入点） |
| 功能范围 | 全部 CTAP 功能 | 仅领导人肖像功能 |

## 安装指南

### 前置条件

1. 已安装 **MT 管理器** (MT Manager)
2. 已有《文明时代 2》原版 APK
3. APK 已包含 `team.rainfall.finality.luminosity2` 注入框架（CTAP 基于此框架运行）

### 方式一：懒人包安装（推荐）

适用于已安装 CTAP 基础框架、只需升级/替换文件的用户。

**步骤：**

1. 打开 MT 管理器，左侧窗口导航到 `CTAP/lazy/` 目录
2. 右侧窗口打开游戏 APK → 进入 `classes.dex`
3. 在右侧 dex 中找到 `team/rainfall/ctap/` 目录
4. 长按左侧的 `team/` 文件夹 → 选择 **复制**
5. 粘贴到右侧 `classes.dex` 内的 `team/rainfall/` 目录
6. 系统提示覆盖时，选择 **全部覆盖**
7. 保存 dex → 退出 APK 编辑
8. 签名 APK → 安装

**MT 管理器操作示意：**
```
左窗口 (文件系统)                    右窗口 (APK → classes.dex)
CTAP/lazy/team/rainfall/ctap/   →   classes.dex/team/rainfall/ctap/
├── CTAP.smali                         (覆盖)
├── MixinCore.smali                    (覆盖)
├── MixinCFG.smali                     (覆盖)
├── MixinGameCalendar.smali            (覆盖)
├── MixinTouchManager.smali            (覆盖)
├── MixinMapBG.smali                   (覆盖)
├── DateUtil.smali                     (覆盖)
├── NumberShortenUtil.smali            (覆盖)
└── CTAP$$ExternalSyntheticLambda0.smali (覆盖)
```

### 方式二：模组包安装

适用于手动定制注入、只需要领导人肖像功能的用户。

**步骤：**

1. 打开 MT 管理器，左侧导航到 `CTAP/mod/team/rainfall/ctap/`
2. 右侧打开游戏 APK → `classes.dex` → `team/rainfall/ctap/`
3. 将 `MixinCore.smali` 复制到右侧目录，覆盖原文件
4. 保存 dex → 签名 → 安装

### 方式三：从原版 APK 注入（首次安装）

如果你的 APK 还没有 CTAP 注入框架，需要先注入全部文件：

1. 使用 MT 管理器反编译 APK（可选）
2. 将 `CTAP/lazy/team/` 整个目录注入到 `classes.dex` 中
3. 确保注入路径为 `classes.dex/team/rainfall/ctap/`
4. 保存并签名

## 配置指南

### 领导人肖像配置

#### 数据格式

在领导人数据文件（通常是 JSON 或游戏内部数据）中设置 `image` 字段：

```json
{
  "name": "特朗普",
  "image": "trump.png",
  "born": "1946-06-14",
  "era": "现代"
}
```

#### 图片命名规则

| 字段值 | 对应图片文件 |
|--------|-------------|
| `"trump.png"` | `game/leadersIMG/trump_SL.png` |
| `"biden.png"` | `game/leadersIMG/biden_SL.png` |
| `"putin.png"` | `game/leadersIMG/putin_SL.png` |
| `"xi.png"` | `game/leadersIMG/xi_SL.png` |

**规则：** `<image字段名去除扩展名>_SL.png`

#### 图片放置位置

```
APK 根目录/
├── assets/
│   └── game/
│       └── leadersIMG/
│           ├── trump_SL.png      ← 特朗普肖像
│           ├── biden_SL.png      ← 拜登肖像
│           ├── putin_SL.png      ← 普京肖像
│           └── ...
```

#### 图片规格建议

| 属性 | 建议值 |
|------|--------|
| 格式 | PNG（支持透明背景） |
| 尺寸 | 宽度 200-400px，高度自适应 |
| 比例 | 3:4 或 4:5（竖版肖像） |
| 背景 | 透明或纯色（推荐透明） |

## 常见问题

### Q1: 安装后游戏崩溃

**可能原因：**
1. 注入路径错误（smali 文件未放到 `classes.dex/team/rainfall/ctap/`）
2. 缺少注入框架（`team.rainfall.finality.luminosity2` 不存在）
3. dex 保存后未正确签名

**排查方法：**
1. 用 MT 管理器查看 `classes.dex` 中是否存在 `team/rainfall/ctap/` 目录
2. 确认所有 9 个 smali 文件都已注入
3. 重新签名 APK 后安装

### Q2: 领导人肖像不显示

**可能原因：**
1. 领导人数据中没有 `image` 字段
2. `_SL.png` 图片未放置到正确路径
3. 图片文件名与 `image` 字段不匹配

**排查方法：**
1. 检查领导人数据是否包含 `"image": "xxx.png"`
2. 确认 `game/leadersIMG/xxx_SL.png` 文件存在
3. 检查文件名大小写是否完全一致

### Q3: 中文数字没有变成"万/亿"

**可能原因：**
- 游戏语言未设置为中文

**解决方法：**
- 在游戏设置中将语言切换为中文（`LANG_TAG` 需包含 `"cn"`）

### Q4: 更新 MixinCore 后其他功能异常

**说明：**
- `MixinCore.smali` 是领导人肖像功能的唯一新增文件
- 其他 8 个文件是 CTAP 原有功能（中文适配、数字简化等）
- 如果只需要肖像功能，可以只更新 `MixinCore.smali`

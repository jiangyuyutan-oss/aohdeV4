# CTAP Mod 插件

CTAP (Chinese Text Adaptation Plus) Mod 的增强功能。

## 功能

### 领导人悬浮窗肖像

在领导人信息悬浮窗中显示领导人肖像大图。

**使用方法：**
1. 在领导人数据中设置 `image` 字段（如 `"trump.png"`）
2. 将悬浮窗大图命名为 `<文件名>_SL.png`（如 `trump_SL.png`）
3. 放置到 `game/leadersIMG/` 目录下

**示例：**
```
game/leadersIMG/trump_SL.png
```

**悬浮窗布局：**
```
[行1] 领导人名称
[行2] 领导人肖像 (_SL.png) ← 新增
[行3] 旗帜 + 宗教图标
[行4] 出生日期 + 时代信息
...
```

## 替换方式

### 懒人包 (lazy/)
直接替换 APK 中 `team/rainfall/ctap/` 目录下对应文件。

### 模组包 (mod/)
仅包含修改的文件，适合手动注入。

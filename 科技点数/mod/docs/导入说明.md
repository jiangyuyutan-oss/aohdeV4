# 科技点 Mod - 快速导入

## 导入步骤

### 第一步：复制文件到 APK

1. MT 管理器**双窗口模式**
2. 左窗口打开本仓库的 `mod/` 目录
3. 右窗口打开游戏 APK → `classes.dex` → `age/of/civilizations2/jakowski/lukasz/GameValues/`
4. 左窗口选中 `TechConfigLoader.smali` → 复制 → 右窗口粘贴

### 第二步：修改一处代码

右窗口打开 `GV_Technology.smali` → 文本编辑器 → 搜索 `ADD_TECH_POINTS_PER_CLICK`

在找到这行**之后**、`return-void` **之前**，插入一行：

```smali
    invoke-static {p0}, Lage/of/civilizations2/jakowski/lukasz/GameValues/TechConfigLoader;->applyModConfig(Lage/of/civilizations2/jakowski/lukasz/GameValues/GV_Technology;)V
```

保存 → 是（回编译）

### 第三步：签名安装

退出 APK → 点击 APK → 功能 → APK 签名 → 安装

### 第四步：使用

游戏根目录创建 `mods/technology.json`：

```json
{
  "MAX_POINTS_RESEARCH": 60,
  "PER_POINT_RESEARCH": 0.85
}
```

## 一行命令导入

如果你会用 MT 管理器的终端：

```bash
cp /storage/emulated/0/Download/mod/age/of/civilizations2/jakowski/lukasz/GameValues/TechConfigLoader.smali /storage/emulated/0/Download/classes.dex/age/of/civilizations2/jakowski/lukasz/GameValues/
```

# 自定义图片 (customImages)

为游戏添加自定义 UI 图片支持，包括电台音乐图标和下一回合按钮装饰图片。

## 功能

- **电台音乐图标**: 将音乐列表中的图标替换为 `music.png`
- **下一回合按钮**: 在按钮右下角显示 `nextturn.png` 装饰图片
- **已包含 GameValues 移动端解锁**: 此版本的 Menu_InitGame.smali 已同时移除了 `getIsDesktop()` 限制

## 前置要求

准备以下图片文件：
- `music.png` - 电台音乐列表图标（建议尺寸：32x32）
- `nextturn.png` - 下一回合按钮装饰图片（建议尺寸：根据按钮大小调整）

将图片放入游戏 assets 的 `UI/icons/` 目录下。

## 使用方式

### 方式一：懒人包 (lazy/)

直接替换以下 4 个 smali 文件：

```
age/of/civilizations2/jakowski/lukasz/Images.smali
age/of/civilizations2/jakowski/lukasz/Menus/Menu_InitGame.smali
age/of/civilizations2/jakowski/lukasz/Menus/SFX/Menu_InGame_Playlist.smali
age/of/civilizations2/jakowski/lukasz/Button/Game/Button_GameNextTurn.smali
```

### 方式二：模组包 (mod/)

手动修改以下文件：

1. **Images.smali** - 添加两个字段和对应方法：
   - `musicImg:I` - music.png 图片 ID
   - `nextTurnImg:I` - nextturn.png 图片 ID
   - `getMusicImg()I` / `setMusicImg(I)V`
   - `getNextTurnImg()I` / `setNextTurnImg(I)V`

2. **Menu_InitGame.smali** - 在图片加载步骤中添加：
   - 加载 `UI/icons/music.png` 并赋值给 `Images.musicImg`
   - 加载 `UI/icons/nextturn.png` 并赋值给 `Images.nextTurnImg`

3. **Menu_InGame_Playlist.smali** - 修改音乐列表项图标：
   - 将 `Images.arrow` 改为 `Images.musicImg`

4. **Button_GameNextTurn.smali** - 添加图片绘制：
   - 在 `drawButtonBGE` 方法末尾绘制 `nextturn.png`

## 修改清单

| 文件 | 修改类型 | 说明 |
|------|---------|------|
| `Images.smali` | 新增字段+方法 | 添加 musicImg、nextTurnImg 字段及 getter/setter |
| `Menu_InitGame.smali` | 新增代码 | 加载 music.png 和 nextturn.png |
| `Menu_InGame_Playlist.smali` | 修改引用 | 音乐列表图标改为 musicImg |
| `Button_GameNextTurn.smali` | 新增代码 | 在按钮右下角绘制 nextturn.png |

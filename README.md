# AoC2 Mod 插件集

本分支包含 Age of Civilizations 2 (文明时代2) 的游戏 Mod 插件。

**注意：本分支仅包含 Mod 修改文件，不包含游戏完整源代码。游戏源代码请切换到 `main` 分支。**

## 插件列表

### 战斗配置 (战斗配置/)

通过 JSON 配置文件动态修改战斗相关参数。

**功能：**
- 科技等级攻击/防御加成
- 战斗省份经济损失
- 战斗省份人口损失
- 首都攻击/防御加成
- 要塞/城堡/堡垒防御加成
- 瞭望塔防御加成
- 军事专家属性

**使用方式：**
- `lazy/` - 懒人包：直接替换文件
- `mod/` - 模组包：手动注入到 APK

### 科技点数 (科技点数/)

通过 JSON 配置文件动态修改科技研究相关参数。

**功能：**
- 科技研究成本
- 科技研究速度
- 科技等级上限

**使用方式：**
- `lazy/` - 懒人包：直接替换文件
- `mod/` - 模组包：手动注入到 APK

## 文件结构说明

每个插件包含两个子目录：

| 目录 | 说明 |
|------|------|
| `lazy/` | 懒人包 - 包含修改后的完整 .smali 文件和配置文件，直接替换即可 |
| `mod/` | 模组包 - 仅包含新增/修改的文件和模板，适合手动注入 |

## 快速开始

1. 选择需要的插件目录
2. 按照 `使用说明.md` 或 `导入指南.md` 操作
3. 签名并安装 APK

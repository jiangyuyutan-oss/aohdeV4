# 战斗配置 Mod - 导入指南

## 文件结构

```
mod/
├── age/of/civilizations2/jakowski/lukasz/GameValues/
│   └── BattleConfigLoader.smali    # 新增：配置加载器
└── template/
    └── battle.json                 # 模板：默认配置文件
```

## 导入步骤

### 1. 修改 GameValues.smali

打开原 APK 的 `GameValues.smali`，找到 `<clinit>` 方法的末尾（`return-void` 之前），添加：

```smali
invoke-static {}, Lage/of/civilizations2/jakowski/lukasz/GameValues/BattleConfigLoader;->applyModConfig()V
```

**原始代码：**
```smali
    .line 210
    const/16 v0, 0x12

    sput v0, Lage/of/civilizations2/jakowski/lukasz/GameValues/GameValues;->DEFAULT_FONT_SIZE:I

    return-void
.end method
```

**修改后：**
```smali
    .line 210
    const/16 v0, 0x12

    sput v0, Lage/of/civilizations2/jakowski/lukasz/GameValues/GameValues;->DEFAULT_FONT_SIZE:I

    invoke-static {}, Lage/of/civilizations2/jakowski/lukasz/GameValues/BattleConfigLoader;->applyModConfig()V

    return-void
.end method
```

### 2. 导入 BattleConfigLoader.smali

将 `age/of/civilizations2/jakowski/lukasz/GameValues/BattleConfigLoader.smali` 复制进 APK 的 `classes.dex` 对应目录。

### 3. 导入配置文件

将 `template/battle.json` 复制到 APK 的 `assets/mods/` 目录下（如无则创建）。

### 4. 签名安装

## 配置优先级

| 位置 | 优先级 | 说明 |
|------|--------|------|
| 外部存储 `Android/data/<包名>/files/mods/battle.json` | 最高 | 可随时修改，无需重新打包 |
| APK 内部 `assets/mods/battle.json` | 默认 | 打包时的默认配置 |

## 注意事项

1. 配置文件中不存在的键会使用游戏默认值
2. 修改配置文件后需重启游戏生效
3. 外部存储配置文件优先级高于 APK 内部配置
